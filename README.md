# Mortal Aura Vercel Deploy Package

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Vercel
- Import this folder as a Vite project
- Build command: `npm run build`
- Output directory: `dist`

## Admin editor
Default visitors do not see the editor.

Open the admin panel by either:
- pressing `Ctrl/⌘ + Shift + A`
- or using a build/runtime version that preserves your local admin mode

## Important
The uploaded file provided in chat was not a pure config JSON; it contained a full React code dump.
